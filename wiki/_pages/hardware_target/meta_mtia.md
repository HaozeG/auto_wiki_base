# Meta MTIA

Meta MTIA (Meta Training and Inference Accelerator, v3 generation referenced) is Meta's custom-designed AI accelerator chip deployed alongside NVIDIA and AMD GPUs to serve Meta's production advertising and recommendation ranking infrastructure, which runs more than 1500 distinct multi-stage ranking models across a fleet processing hundreds of trillions of ad-ranking inferences daily. Architecturally, MTIA differs from GPU accelerators through a custom on-chip SRAM subsystem optimized for recommendation-model inference access patterns rather than a large shared cache hierarchy, and it is programmed primarily through Meta's Triton compiler stack: Triton source lowers through a platform-independent Triton-MLIR representation into a hardware-specific TritonMTIA-MLIR dialect, then through LLVM-MLIR/LLVM-IR, and finally through codegen that emits a native RISC-V binary for execution on MTIA — making RISC-V the target instruction-set architecture for Meta's Triton-generated MTIA kernels, alongside PTX/CUBIN for NVIDIA and AMDGCN/HSACO for AMD GPUs in the same compilation pipeline. MTIA is one of the three heterogeneous hardware targets (alongside NVIDIA and AMD GPUs) that Meta's KernelEvolve agentic kernel-coding framework generates and optimizes production Triton kernels for.

## Key Claims

- MTIA is Meta's custom AI accelerator chip, deployed in production alongside NVIDIA and AMD GPUs to serve recommendation ranking models at data-center scale.
- Meta's Triton compiler stack targets MTIA through a dedicated TritonMTIA-MLIR dialect that lowers through LLVM-MLIR/LLVM-IR to a native RISC-V binary, distinct from the PTX/CUBIN path for NVIDIA and the AMDGCN/HSACO path for AMD GPUs.
- MTIA features a custom on-chip SRAM subsystem with distinct on-chip and off-chip bandwidth profiles, architecturally different from NVIDIA's large L2-cache-plus-Infinity-Cache-style hierarchies.
- Kernels generated by Meta's KernelEvolve framework for MTIA-relevant/MTIA-training workloads reported speedups over PyTorch baselines including MapId transform 4.1x, MergeBucketizedDense/MBDT transform 9.3x, Batch Event Truncate 9.8x, and RMSNorm 2D backward 17x.
- MTIA is one of Meta's three production AI-accelerator hardware generations (alongside NVIDIA and AMD GPUs) targeted by Meta's KernelEvolve agentic kernel-coding framework for automated Triton kernel generation and optimization of deep learning recommendation model (DLRM) training and inference.

## Relationships
