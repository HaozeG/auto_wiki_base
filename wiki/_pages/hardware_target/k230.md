# K230

The K230 is a system-on-chip (SoC) from Canaan Inc. (Kendryte series) designed for AIoT applications, integrating two RISC-V C908 cores and a dedicated Knowledge Process Unit (KPU). The chip supports RISC-V 64GCB instruction set on one core (max 800 MHz) and RISC-V Vector Extension 1.0 on the high-performance core (max 1.6 GHz). Memory hierarchy includes 32 KB L1 I-cache and D-cache per core, 128 KB L2 cache on the first core and 256 KB on the second, plus a 2 MB system SRAM. The KPU provides INT8 and INT16 multi-precision AI inference, supporting typical networks like ResNet50 at 85 fps (INT8). Additional accelerators include a 2D DMA engine, an AI 2D engine for affine/crop/resize, a 2.5D GPU, and a depth processing unit (DPU) for 3D structured light depth calculation. The chip supports multiple peripherals and is targeted at smart hardware such as door locks, cameras, drones, and robots.

## Key Claims

- Dual-core RISC-V: one at 800 MHz (64GCB) and one at 1.6 GHz (RVV 1.0).
- KPU supports INT8/INT16 with weight sparse compression.
- Typical KPU performance (INT8): ResNet50 ≥85fps, Mobilenet_v2 ≥670fps, YoloV5S ≥38fps.
- Supports TensorFlow, PyTorch, TFLite, PaddlePaddle, ONNX with quantization accuracy loss <1%.
- 2D GDMA engine supports mirror, rotation (90/180/270), up to 64Kx64K resolution.
- AI 2D engine supports affine transformation, crop, resize, padding, and shift.
- DPU enables 3D structured light depth calculation up to 1920x1080 resolution.

## Optimization-Relevant Details

- ISA/profile: RISC-V 64GCB (CPU0), RISC-V Vector Extension 1.0 (CPU1).
- Vector/matrix/accelerator support: CPU1 128-bit vector unit, KPU (convolution acceleration), AI 2D engine, DPU, 2.5D GPU.
- Memory/cache/TLB/DMA: L1 I-cache 32KB, D-cache 32KB; L2 cache 128KB (CPU0), 256KB (CPU1); system SRAM 2MB; GDMA with configurable AXI outstanding.
- Compiler/toolchain support: supports TensorFlow/PyTorch/ONNX; quantization toolchain; profiling tools.
