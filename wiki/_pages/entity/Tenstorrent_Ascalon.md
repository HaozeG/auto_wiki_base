# Tenstorrent Ascalon

The Tenstorrent Ascalon is a processor core based on the RISC-V instruction set, developed by Tenstorrent, a next-generation computing company that builds computers for artificial intelligence. Imperas Software, a leader in RISC-V simulation solutions, has collaborated with Tenstorrent to develop a simulation model of the Ascalon core, available as part of the Imperas RISC-V model library. This model supports many time-critical aspects of system-on-chip (SoC) development, including early-stage software development, firmware and driver development, and architectural exploration. The model matches the Ascalon's configuration space, supporting multi-core and many-core options, and offers fast simulation performance to help address time-to-market and time-to-volume factors for SoC developers adopting Tenstorrent IP.

## Key Claims

- Imperas Software has developed a simulation model of the Tenstorrent Ascalon RISC-V processor core.
- The model is available as part of the Imperas RISC-V model library.
- The model supports multi-core and many-core configuration options.
- It provides fast simulation performance for software development and architectural exploration.
- The collaboration aims to accelerate time-to-market for SoC developers using Tenstorrent IP.
