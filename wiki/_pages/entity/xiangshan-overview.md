# XiangShan

XiangShan is an open-source, high-performance RISC-V processor designed to be industry-competitive and serve as infrastructure for architecture research. Launched in June 2020 by the XiangShan project, it has produced two major generations codenamed YQH and NH. The processor implements the RV64GCBK ISA extension set and is a superscalar out-of-order design featuring a high-throughput frontend with an advanced branch predictor, a six-width aggressive out-of-order execution engine, a high-bandwidth load/store unit, and a highly configurable cache system. It is written in Chisel, a high-level hardware description language that improves readability and maintainability. According to the project, the latest version of XiangShan achieves the highest performance of any open-source RISC-V processor. The development process is supported by the Minjie platform, an open-source agile development infrastructure that integrates tools for functional verification, performance evaluation, and simulation flow. The processor has been taped out and has a demonstrated FPGA prototype.
