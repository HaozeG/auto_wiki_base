# CHIPS Alliance

CHIPS Alliance (Common Hardware for Interfaces, Processors and Systems Alliance) is an open source hardware and software development organization hosted under the Linux Foundation. It aims to accelerate hardware development by fostering collaborative creation and sharing of high-quality open source designs relevant to ASICs and FPGAs. The Alliance develops and hosts a range of reusable intellectual property (IP) blocks including RISC-V processor cores, neural network accelerator cores, interconnect IP (both physical and logical protocols), and open source electronic design automation (EDA) tools for design, verification, and simulation.

## Key Claims

- Member of the Linux Foundation.
- Develops high-quality, open source hardware designs for ASICs and FPGAs.
- Creates reusable IP blocks, including RISC-V cores and neural network accelerator cores.
- Develops interconnect IP covering physical and logical protocols.
- Provides open source EDA tools for design, verification, and simulation.
- Lowers cost of silicon development through shared resources and a collaborative environment.
