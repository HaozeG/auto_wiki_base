# CHIPS Alliance

CHIPS Alliance (Common Hardware for Interfaces, Processors and Systems Alliance) is an open-source hardware and software development organization hosted under the Linux Foundation. It aims to accelerate hardware development by fostering collaborative creation and sharing of high-quality open source designs relevant to ASICs and FPGAs. The Alliance develops and hosts a range of reusable intellectual property (IP) blocks including RISC-V processor cores, neural network accelerator cores, interconnect IP (both physical and logical protocols), and open source electronic design automation (EDA) tools for design, verification, and simulation. The alliance hosts multiple open-source projects, including Chisel (a hardware design language written in Scala), Rocket Chip (a System-on-Chip generator), Verible (a suite of SystemVerilog developer tools covering parsing, linting, formatting, and language server support), riscv-dv (a random instruction generator for RISC-V processor verification), Cores-VeeR-EH1 (a VeeR EH1 core), FIRRTL (a Flexible Intermediate Representation for RTL), and the Caliptra subsystem (including Root of Trust IP, RTL, firmware, and software). The alliance has over 122 repositories, including VeeR EL2 core, Caliptra subsystem software, and Verilator (forked). By creating an open and collaborative environment, the alliance aims to lower the cost of silicon development through shared resources. The repositories are primarily licensed under Apache-2.0, with some under ISC.
