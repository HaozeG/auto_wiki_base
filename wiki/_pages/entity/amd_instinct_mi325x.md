# AMD Instinct MI325X

The AMD Instinct MI325X is a data center AI accelerator announced by AMD at Computex 2024 as part of the updated AMD Instinct annual roadmap. It is designed to deliver high memory capacity and bandwidth for large AI model inference and training, featuring 288 GB of HBM3E memory with 6 TB/s of memory bandwidth. The MI325X uses the same industry standard Universal Baseboard server design as the AMD Instinct MI300 series, enabling easy integration into existing server infrastructure. It is expected to be available in Q4 2024. AMD claims the MI325X provides 2x better memory capacity and 1.3x better memory bandwidth than the competition, as well as 1.3x better compute performance. The accelerator continues the AMD Instinct family's focus on AI workloads and is part of an annual cadence that includes the MI350 series (based on CDNA 4, 2025) and MI400 series (CDNA Next, 2026).

## Key Claims

- 288 GB HBM3E memory with 6 TB/s memory bandwidth.
- Same Universal Baseboard server design as the AMD Instinct MI300 series.
- General availability in Q4 2024.
- AMD claims 2x memory capacity, 1.3x memory bandwidth, and 1.3x compute performance vs competition (unnamed).
- Part of an annual product cadence spanning MI300, MI325X, MI350, and MI400 series.

## Relationships

- The MI325X directly precedes the [[amd_cdna_4]]-based MI350 series in AMD's Instinct roadmap, using a similar server platform but with CDNA 3 architecture and HBM3E memory.
