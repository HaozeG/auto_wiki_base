# XuanTie C908

The T-Head XuanTie C908 is a 64-bit RISC-V core developed by Alibaba's T-Head Semiconductor, announced in November 2022 and targeting mid-range AIoT (Artificial Intelligence of Things) applications. It features a dual-issued, 9-stage in-order pipeline with support for the RISC-V Vector extension (RVV) acceleration. The core adopts the RV64GCB[V] instruction set and complies with the RVA22 profile, enabling compatibility with Android and other rich operating systems. According to T-Head, the C908's performance lies between the earlier C906 (introduced 2020) and C910 (2019) cores, positioning it as a balanced option for AIoT workloads that require vector processing without the complexity and power of high-end out-of-order cores.

## Key Claims

- 64-bit RISC-V core with dual-issued, 9-stage in-order pipeline.
- Supports RISC-V Vector extension (RVV) acceleration.
- Adopts RV64GCB[V] instruction set and complies with the RVA22 profile for rich OS compatibility.
- Targets mid-range AIoT applications.
- Performance between the C906 and C910 cores.
- Developed by Alibaba's T-Head Semiconductor.
- Announced November 2, 2022.
