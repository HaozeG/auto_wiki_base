# Kendryte K510

The K510 is a fully open-source high-precision AI edge inference chip developed by Canaan Inc. It integrates a dual-core RISC-V architecture CPU and the proprietary KPU2.0 AI accelerator, enabling edge-side inference for a variety of artificial intelligence workloads without dependence on proprietary ecosystems. The chip fully supports the Linux operating system and the RISC-V P extension for enhanced vector processing capabilities. It features a triple HD camera setup, expanding the range of possible vision applications such as multi-view object detection and depth sensing. The associated K510 SDK is hosted on GitHub and provides APIs and usage instructions for software development. Additionally, the DongshanPI-Vision development board, launched by 100ask.net, offers a comprehensive set of Linux+AI tutorials based on the K510, making it accessible for educational and prototyping purposes.
