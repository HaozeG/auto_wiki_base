# NVIDIA Blackwell Platform

The NVIDIA Blackwell Platform is a full-stack computing platform designed by NVIDIA for AI and accelerated computing workloads, announced at Hot Chips 2024. It integrates multiple chips including the Blackwell GPU, Grace CPU, BlueField data processing unit, ConnectX network interface card, NVLink Switch, Spectrum Ethernet switch, and Quantum InfiniBand switch, all orchestrated by NVIDIA CUDA software. The platform aims to deliver unprecedented performance and energy efficiency for generative AI and large language model inference, with the multi-node GB200 NVL72 system connecting 72 Blackwell GPUs and 36 Grace CPUs via liquid cooling. The NVIDIA Quasar Quantization System, combining algorithmic innovations and Blackwell's second-generation Transformer Engine, supports high-accuracy low-precision models.

## Key Claims

- The NVLink interconnect provides all-to-all GPU communication for record high throughput and low-latency generative AI inference.
- The GB200 NVL72 system acts as a unified system delivering up to 30x faster inference for LLM workloads, enabling real-time trillion-parameter models.
- The Quasar Quantization System combines algorithmic innovations, software libraries, and Blackwell's Transformer Engine to support high accuracy on low-precision models.
- NVIDIA presented hybrid liquid-cooling solutions (retrofit, direct-to-chip, immersion) for data centers, as part of the COOLERCHIPS program, using Omniverse digital twins for cooling optimization.
- AI agents and LLMs are being developed for chip design, assisting with code generation, debugging, and autonomous tool use.
