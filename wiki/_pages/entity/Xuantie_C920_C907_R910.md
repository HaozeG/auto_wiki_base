# Xuantie C920, C907, R910 Processors

The Pingtouge Xuantie C920, C907, and R910 are a family of RISC-V processors released by Alibaba's DAMO Academy in November 2023, targeting diverse application domains including autonomous vehicles, artificial intelligence, enterprise hard drives, and network communications. The C907 processor is notable for implementing matrix operation (Matrix) expansion for the first time in the Xuantie family, while the C920 meets the Vector 1.0 standard for enhanced vector processing. The R910 is a real-time processor designed for latency-sensitive applications. The launch represents a significant step in commercializing RISC-V technology, with plans for the first large-scale RISC-V Android device in 2024 and the flagship Xuantie C930 processor announced for release within the same year.

## Key Claims

- Released in November 2023 by Alibaba's DAMO Academy, targeting autonomous vehicles, AI, enterprise hard drives, and network communications.
- C907: First Xuantie processor to implement matrix operation (Matrix) expansion for AI acceleration.
- C920: Supports the RISC-V Vector 1.0 standard for efficient vector processing.
- R910: Real-time processor optimized for latency-sensitive applications.
- First large-scale RISC-V Android device announced for 2024, leveraging the Xuantie ecosystem.
- Xuantie C930 flagship processor announced for release in 2024, extending the Xuantie family.
