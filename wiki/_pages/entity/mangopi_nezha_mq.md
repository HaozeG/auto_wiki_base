# MangoPi-Nezha MQ

The MangoPi-Nezha MQ is a 40 x 40mm single-board computer (SBC) launched in March 2022 by MangoPi, priced at $39. It is based on the Allwinner F133-A system-on-chip (SoC), which features a single-core 1 GHz RISC-V processor. The board includes 64 MB of DDR2 memory and provides a range of I/O options: two USB Type-C ports, WiFi connectivity, a MIPI-DSI display interface, RGB touch support, a DVP camera interface, general-purpose I/O (GPIO) pins, and audio input/output. It runs Linux or the RT-Smart real-time operating system. The board is notable as an affordable, compact RISC-V development platform.

## Key Claims

- Launched at $39 with preorders starting March 2022.
- Dimensions: 40 x 40 mm.
- SoC: Allwinner F133-A with a single-core 1 GHz RISC-V CPU.
- Memory: 64 MB DDR2.
- Connectivity: 2x USB Type-C, built-in WiFi.
- Display interfaces: MIPI-DSI, RGB touch.
- Camera: DVP interface.
- GPIO and audio I/O included.
- Runs Linux and RT-Smart operating systems.
