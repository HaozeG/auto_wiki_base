# Apple M5

The Apple M5 is a system-on-a-chip designed by Apple Inc., launched on October 15, 2025, powering the 14-inch MacBook Pro, iPad Pro, and Apple Vision Pro. Built on TSMC's third-generation 3nm process (N3P), it introduces Neural Accelerators embedded in each GPU core, a first for Apple silicon. The M5 features up to 10 GPU cores with embedded Neural Accelerators, delivering a 4x peak GPU compute performance increase for AI workloads compared to the M4. It includes a 16-core Neural Engine for energy-efficient AI inference and 153GB/s unified memory bandwidth (30% increase over M4), which reduces memory bottlenecks for local LLM inference. The M5 Pro and M5 Max variants are expected in early 2026. Apple's MLX benchmarks show the M5 achieving time-to-first-token generation under 10 seconds for dense 14B parameter models and under 3 seconds for 30B MoE models on a laptop.

## Key Claims

- Launched October 15, 2025 on TSMC N3P process.
- First Apple silicon with Neural Accelerators embedded in each GPU core.
- 4x peak GPU compute performance for AI compared to M4.
- 16-core Neural Engine.
- 153GB/s unified memory bandwidth (30% increase over M4).
- MLX benchmarks: time-to-first-token <10s for dense 14B models, <3s for 30B MoE models.
- Capable of running Llama 2 7B quantized models locally with usable response times.
