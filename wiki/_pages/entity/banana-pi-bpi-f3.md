# Banana Pi BPI-F3

The Banana Pi BPI-F3 is an industrial-grade RISC-V development board powered by the SpacemiT K1 octa-core processor, which delivers 2.0 TOPs of AI computing power. The board supports up to 16GB of LPDDR4 RAM and offers flexible onboard storage options including 8/16/32/128GB eMMC. Connectivity features include dual Gigabit Ethernet ports, four USB 3.0 ports, a PCIe M.2 interface, and HDMI output, making it suitable for edge computing, industrial control, and AI applications. The BPI-F3 is designed to be a versatile platform for RISC-V development, leveraging the RVA22-compliant SpacemiT X60 cores.

## Key Claims

- Industrial-grade RISC-V development board.
- SpacemiT K1 8-core RISC-V chip with 2.0 TOPs AI.
- Supports 2/4/8/16GB LPDDR4 and 8/16/32/128GB eMMC.
- Dual GbE, 4x USB 3.0, PCIe M.2, HDMI.
- Designed for edge computing and industrial control.
