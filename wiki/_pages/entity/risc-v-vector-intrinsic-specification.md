# RISC-V Vector Intrinsic Specification

The riscv-rvv-intrinsic-doc repository is the official working draft for the RISC-V Vector Extension (RVV) intrinsic API specification. Maintained under the riscv-non-isa organization on GitHub, it provides specification documents in the doc/ directory and auto-generated intrinsic prototypes in the auto-generated/ directory, produced by scripts in the rvv-intrinsic-generator/ subdirectory. The intrinsics specification is versioned, with the latest release being v1.0, which is supported by Clang 19 and GCC 14 toolchains. The repository includes a CI/CD automation suite via GitHub Actions and an API testing framework for verifying compiler support. Developers can use the generated header files to write vector-accelerated programs using standard C intrinsics for RVV.

## Key Claims

- The repository contains the working draft for the RISC-V Vector intrinsic API specification.
- Intrinsic prototypes are auto-generated by scripts in rvv-intrinsic-generator/.
- The latest specification version is v1.0.
- Clang 19 and GCC 14 support the v1.0 version.
- The repository includes CI/CD automation and testing infrastructure.
