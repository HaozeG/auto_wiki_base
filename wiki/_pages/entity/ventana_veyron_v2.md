# Ventana Veyron V2

The Ventana Veyron V2 is a 192-core server processor developed by Ventana Micro Systems that implements the RISC-V instruction set architecture. Announced in late 2023, the Veyron V2 represents the second generation of Ventana's server-class processor line, following the Veyron V1 which debuted in 2022 but failed to achieve significant market adoption. The V2 design employs a modular chiplet architecture that allows up to 192 RISC-V cores to be integrated into a single processor package, and it supports the addition of specialized accelerators to handle compute-intensive workloads. This modular approach is intended to enable flexibility in addressing diverse server workloads ranging from cloud computing to AI inference. The Veyron V2 is positioned by Ventana to compete with established x86 server processors from Intel and AMD, as well as with other RISC-V server efforts. Its design incorporates features expected to be standard in next-generation RISC-V server chips, including support for vector extensions and chiplet interconnect protocols.
