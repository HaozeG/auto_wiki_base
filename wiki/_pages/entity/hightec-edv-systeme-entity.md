# HighTec EDV-Systeme

HighTec EDV-Systeme GmbH is a German company specializing in safety-qualified compiler solutions for embedded systems, particularly in the automotive and industrial sectors. Founded in 1982, HighTec offers ISO 26262 ASIL D certified toolchains built on LLVM technology, supporting multiple architectures including Arm, TriCore, Renesas RH850, RISC-V, and Power Architecture. The company provides a comprehensive development platform with Rust and C/C++ compilers, the PXROS-HR real-time operating system qualified to ASIL D and IEC 61508 SIL 3, and tool qualification kits to accelerate certification activities. HighTec collaborates with silicon partners to optimize compilers for new architectures and has partnered with SiFive to enable safety-ready RISC-V software development for automotive and industrial applications.

## Key Claims

- HighTec's Rust and C/C++ toolchain is qualified according to ISO 26262 up to ASIL D and certified to the cybersecurity standard ISO 21434.
- The toolchain is built on advanced LLVM technology and supports SiFive's RISC-V IP with tuned performance enhancements.
- The toolchain enables hybrid development combining legacy C/C++ code with modern Rust.
- HighTec provides a comprehensive Qualification Kit to accelerate and simplify tool qualification for safety and security certification.
- HighTec offers the PXROS-HR real-time operating system, certified to ISO 26262 ASIL D and IEC 61508 SIL 3.
- HighTec's compilers cover multiple architectures: Arm, TriCore/AURIX/TRAVEO, Renesas RH850, RISC-V, Power Architecture, and Embedded AI accelerators.
- The collaboration with SiFive aims to reduce integration effort and shorten time to production for safety-relevant RISC-V designs.
