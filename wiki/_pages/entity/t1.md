# T1

T1 (Torrent-1) is an open-source RISC-V Vector (RVV) processor implementation developed under the CHIPS Alliance. Inspired by the Cray X1 vector machine, T1 adopts a lane-based microarchitecture that emphasizes intensive chaining support and uses SRAM-based vector register files (VRFs) for data storage. It supports the standard RISC-V vector extensions Zve32f and Zve32x, and its vector length (VLEN) and datapath width (DLEN) are scalable up to 64K elements, pushing the boundaries of the RISC-V vector architecture. T1 ships with default features such as multiple lanes, chaining, and a large load-store unit (LSU) outstanding capability, and it can also serve as a general platform for memory-mapped I/O (MMIO) domain-specific accelerators (DSAs). The design is implemented in the Chisel hardware construction language and a software emulator, T1Emulator, is provided for testing and development. The scalar portion of T1 uses a forked version of the Rocket Core but is designed to be replaceable with any other RISC-V scalar CPU. Currently, T1 only supports bare-metal program loading and execution.
