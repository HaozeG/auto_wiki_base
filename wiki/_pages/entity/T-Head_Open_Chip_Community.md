# T-Head Open Chip Community (OCC)

The T-Head Open Chip Community (OCC) is a cloud-based platform provided by T-HEAD Semiconductor Co., Ltd., a wholly owned subsidiary of the Alibaba Group, serving as the infrastructure for T-Head chip ecosystem development. It offers a development center, chip creation space, and technology tribes where developers and chip partners can access aggregated application resources, tools, and documentation. OCC aims to facilitate hardware-software integration by enabling application-driven chip definition and providing full-stack technical support for agile chip development. The platform includes training camps, download resources for tools like GCC binaries supporting RISC-V Vector 1.0, and community interaction blocks. It targets both chip enterprises seeking precise chip definition and developers seeking ecosystem support.

## Key Claims

- OCC is a cloud-based platform that acts as the infrastructure for T-Head (平头哥) chip ecosystem development.
- It provides a development center, chip creation space, and technology tribes for community interaction.
- The platform aggregates application resources to support chip partners in building hardware-software integrated ecosystems.
- OCC uses an application-driven approach to chip definition and provides full-stack technical support for agile development.
- It offers training camps and download resources, including GCC binaries for RISC-V Vector 1.0 support.
- The community is part of the Alibaba Cloud Developer Ecosystem.
