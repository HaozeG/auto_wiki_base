# Rocket Chip Generator

Rocket Chip is an open-source, parameterized RISC-V SoC generator developed at UC Berkeley's Architecture Research group, written entirely in the Chisel hardware construction language. First published in 2016, Rocket Chip produces synthesizable RTL for a complete SoC including the Rocket in-order scalar CPU core, an optional BOOM out-of-order core, a coherent multi-level cache hierarchy, a TileLink interconnect fabric, IEEE-compliant floating-point units, and support for virtual memory via an MMU. The generator philosophy — composing chip-building libraries rather than handwriting RTL — established the template for modern agile hardware development. Rocket Chip is now maintained under the ChipsAlliance umbrella and serves as the foundation for the Gemmini neural-network accelerator, the FireSim FPGA simulation platform, and numerous academic and commercial RISC-V tapeouts globally.

## Key Claims

- Rocket Chip generates complete RISC-V SoC RTL (cores, caches, interconnect, peripherals) from a single parameterized Chisel description.
- The Rocket scalar core is a 5-stage in-order pipeline implementing RV64GC with optional extensions; it targets Linux-capable application-class workloads.
- BOOM (Berkeley Out-of-Order Machine) is co-generated within the same framework, sharing the same cache hierarchy and memory system.
- The TileLink cache-coherent interconnect generated by Rocket Chip supports multi-core configurations with coherent shared L2 caches.
- Gemmini systolic-array ML accelerator attaches to Rocket Chip via the RoCC (Rocket Custom Coprocessor) interface.
- FireSim FPGA simulation platform uses Rocket Chip RTL directly for cycle-accurate datacenter-scale simulation on AWS F1 instances.
- Rocket Chip is hosted at github.com/chipsalliance/rocket-chip under a permissive Apache 2.0 license after transfer from Berkeley to ChipsAlliance.
