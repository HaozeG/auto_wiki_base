# MoS2 RISC-V Prototype

The MoS2 RISC-V prototype is a 32-bit microprocessor developed by Chinese scientists, fabricated using two-dimensional molybdenum disulfide (MoS2) as the semiconductor material with an active layer thickness of approximately one nanometer. The chip integrates 5,900 MoS2 transistors and implements the full RV32I instruction set using a complete standard cell library built on 2D semiconductor technology. This milestone represents the most complex 2D-material-based semiconductor chip reported to date, demonstrating a potential pathway beyond conventional silicon scaling for RISC-V processors. The work was published in Nature and has been covered by multiple news outlets, highlighting the growing interest in non-silicon materials for ultraminiaturized logic circuits.

## Key Claims

- Implements a full 32-bit RISC-V instruction set (RV32I).
- Uses two-dimensional MoS2 as the semiconductor channel material.
- Active semiconductor layer thickness is approximately one nanometer.
- Contains 5,900 MoS2 transistors on the chip.
- Includes a complete standard cell library for 2D semiconductor technology.
- Reported as the world's most complex 2D semiconductor chip.
