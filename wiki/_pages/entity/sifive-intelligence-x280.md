# SiFive Intelligence X280

The SiFive Intelligence X280 is a high-performance 64-bit RISC-V vector processor designed for AI and machine learning workloads. It integrates a 512-bit vector register length and a 256-bit Vector ALU and Load/Store architecture, implementing the RISC-V Vector extension version 1.0. The processor features an 8-stage dual-issue in-order scalar pipeline alongside loosely coupled vector computation pipelines for parallel execution of scalar and vector operations. It supports SiFive Intelligence Extensions, a set of custom instructions that accelerate AI/ML performance-critical operations. The X280 is configurable as a multi-core, multi-cluster design with up to eight cores, offering high-throughput single-thread performance under tight power and area constraints. It includes virtual memory support with up to 48-bit addressing and is aimed at a broad range of applications, including computer vision, mobile ISP, edge AI, and datacenter AI.
