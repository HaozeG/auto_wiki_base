# CV32E40P

The CV32E40P is a small and efficient 32-bit in-order RISC-V core developed by the PULP (Parallel Ultra-Low Power) platform team and later contributed to the OpenHW Group as part of the CORE-V family. It features a 4-stage pipeline and implements the RV32IM[F|Zfinx]C instruction set architecture, which includes integer, multiply/divide, single-precision floating-point (via F or Zfinx extensions), and compressed instructions. Additionally, the core supports the PULP custom extensions, which are designed to improve code density, performance, and energy efficiency for IoT endpoint devices. The core originated as a fork of the OR10N CPU core based on the OpenRISC ISA, and was initially known as RI5CY when it became a RISC-V core in 2016. It was maintained by the PULP platform team until February 2020, when it was contributed to OpenHW Group. The verification environment for the CV32E40P is maintained separately in the core-v-verif repository, and a user manual is available via readthedocs.

## Key Claims

- CV32E40P implements the RV32IM[F|Zfinx]C instruction set architecture, supporting integer, multiply/divide, floating-point, and compressed instructions.
- The core has a 4-stage in-order pipeline.

- The core originated as a fork of the OR10N CPU core (based on OpenRISC) and became the RISC-V core RI5CY in 2016.
