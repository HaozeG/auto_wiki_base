# Sipeed LicheePi 4A

The Sipeed LicheePi 4A is a high-performance single-board computer (SBC) developed by Sipeed Technology, powered by the T-Head TH1520 system-on-chip (SoC) featuring a quad-core RISC-V C910 processor running at 1.85 GHz. The board integrates a 4 TOPS neural processing unit (NPU) based on the C906 core for AI inference tasks and an Imagination BXE-4-32 GPU for graphics rendering. It supports dual 4K display output via HDMI and MIPI, a 4K MIPI camera input, dual Gigabit Ethernet ports, multiple USB interfaces, and runs mainline Linux distributions. The LicheePi 4A uses the Lichee Module 4A core module and was launched as one of the most powerful RISC-V SBCs available as of Q2 2023, with performance claims of approximately twice that of the VisionFive2 and close to the Raspberry Pi 4 when custom instruction set acceleration is enabled.

## Key Claims

- Powered by TH1520 SoC with quad-core RISC-V C910 CPU at 1.85 GHz.
- Integrated NPU delivers up to 4 TOPS (INT8) for AI acceleration.
- GPU is Imagination BXE-4-32.
- Supports dual 4K display (HDMI + MIPI) and 4K MIPI camera input.
- Dual Gigabit Ethernet and multiple USB interfaces.
- Performance claimed to be about 2× that of VisionFive2 and comparable to Raspberry Pi 4 (with custom instructions enabled).
- Uses Lichee Module 4A core module and runs RISC-V Linux.
