# PyAi-k210

The PyAi-k210 is a machine vision platform developed by 01Studio, built around the Kendryte K210 dual-core 64-bit RISC-V processor with an integrated Neural Processing Unit (NPU) for edge AI inference. It uses a Sipeed M1n module with an M.2 interface and is packaged in a form factor similar to the OpenMV camera platform. The board targets machine vision and hearing applications, and supports training neural network models on custom data using a cloud-based platform. 01Studio provides official GitHub repositories with example code and documentation for the platform.

## Key Claims

- Built around the Kendryte K210 dual-core 64-bit RISC-V processor with integrated NPU.
- Uses a Sipeed M1n module interfaced via M.2.
- Form factor similar to the OpenMV camera platform.
- Developed by 01Studio.
- Supports cloud-based training of neural network models on custom data.
- Official GitHub repositories with examples provided by 01Studio.
- Targets edge AI applications including machine vision and hearing.
