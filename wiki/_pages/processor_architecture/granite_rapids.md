# Granite Rapids

Granite Rapids is the codename for the 6th generation Intel Xeon Scalable server processor series, launched on September 24, 2024. Designed by Intel and manufactured on the Intel 3 process, Granite Rapids features up to 128 Redwood Cove P-cores and supports LGA 4710 and LGA 7529 sockets. It includes 12 memory channels for DDR5, 136 PCIe 5.0 lanes, and CXL 2.0 interconnect. The platform is intended for high performance computing workloads, with cache hierarchy of 112 KB L1 (64 KB instructions + 48 KB data), 2 MB L2, and 3 MB L3 per core. Granite Rapids is part of Intel's split P-core/E-core Xeon roadmap, complementing the earlier Sierra Forest E-core processors.

## Key Claims

- Launched September 24, 2024.
- Up to 128 P-cores based on Redwood Cove microarchitecture.
- Fabricated on Intel 3 process.
- Supports x86-64 instruction set with extensions including AES-NI, AVX-512, AMX, and others.
- Memory support: DDR5 with 12 memory channels.
- PCIe 5.0 with 136 lanes.
- CXL 2.0 support.
- Cache: 112 KB L1, 2 MB L2, 3 MB L3 per core.
- Sockets: LGA 4710 and LGA 7529.
- UPI links: up to 6.
- Predecessor: Emerald Rapids; Successor: Diamond Rapids; Variant: Sierra Forest.
