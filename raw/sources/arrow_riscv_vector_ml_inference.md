# Source: Arrow RISC-V Vector Accelerator for ML Inference (arxiv 2107.07169, CARRV 2021)
Fetched: 2026-06-26

## Architecture
- Configurable hardware accelerator implementing subset of RISC-V v0.9 vector ISA extension
- Designed for edge machine learning inference tasks
- FPGA implementation on Xilinx XC7A200T-1SBG484C

## Performance
- 2–78× faster than scalar RISC processor on vector/matrix benchmarks relevant to ML inference
- 20%–99% less energy than baseline scalar processor

## Publication
- Authors: Imad Al Assir, Mohamad El Iskandarani, Hadi Rayan Al Sandid, Mazen A. R. Saghir
- Venue: Fifth Workshop on Computer Architecture Research with RISC-V (CARRV 2021), co-located with ISCA 2021
- Submitted: July 15, 2021

## Notes
- Wide performance variance (2–78×) reflects variability across benchmark types and configurations
